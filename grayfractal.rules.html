grayfractal.rules module¶

Rules for growing fractals.

class grayfractal.rules.Rule[source]¶

Bases: abc.ABC

Abstract base class for fractal growth rules.

A rule defines the new generation as a function of the old generation, as well as capturing the scaling effect this produces.
